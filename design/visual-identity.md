## Asset Checklist Per App

For each of the 10 apps, generate:
1. [ ] App icon (1024x1024)
2. [ ] Splash screen illustration
3. [ ] Background texture/gradient (tileable)
4. [ ] Tileable sacred pattern (for cards/sections)
5. [ ] 3-4 onboarding illustrations
6. [ ] UI accent elements (dividers, card backgrounds)
7. [ ] Empty state illustrations

## Tools
- **Midjourney** — primary image generation ($10/mo)
- **Figma** — layout, icon refinement, export
- **SVG generators** — geometric patterns (can be procedural)
- **Unsplash/Pexels** — supplementary photography (nature, textures)
